Traceroute in Go

This project provides a traceroute library implemented in Go.

The library is part of a larger project inside DNSimple that provides DNS monitoring. Being able to trace how packets are routed from a particular location is quite useful, especially in an Anycast network where you can never be sure of which node a packet will be routed to.

See the README for details on usage.
